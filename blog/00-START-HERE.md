## 🚀 How to Use

### View the Blog
1. Open: `blog/index.html`
2. See all articles in a beautiful grid
3. Filter by category
4. Click any article to read it

### URL Format
```
Blog Listing: /blog/
Article: /blog/articles/index.html?slug=article-slug

Examples:
- /blog/articles/index.html?slug=guide-residency-permits
- /blog/articles/index.html?slug=finding-perfect-apartment
- /blog/articles/index.html?slug=legal-contracts-explained
```

### Add New Articles
1. Edit `/blog/data/blog-data.js`
2. Add article object with metadata
3. Articles automatically appear in listing
4. No additional HTML files needed!

### Customize Styling
1. Edit `/blog/assets/css/blog.css`
2. All styles are well-commented
3. Uses CSS variables for easy customization
4. Mobile breakpoints included

## 🔗 Link Reference

### How to Link from Other Pages

**From main site (index.html, about.html, etc.):**
```html
<!-- Blog listing -->
<a href="blog/">Visit Blog</a>
<a href="blog/index.html">Our Blog</a>

<!-- Specific article -->
<a href="blog/articles/index.html?slug=guide-residency-permits">
  Learn about Residency
</a>
```

**From blog pages:**
```html
<!-- Back to blog listing -->
<a href="./index.html">Back to Blog</a>

<!-- To main site -->
<a href="../index.html">Home</a>

<!-- Between articles -->
<a href="./index.html?slug=other-article">Read Related</a>
```
