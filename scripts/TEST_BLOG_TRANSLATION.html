🔧 Quick Troubleshooting

If articles don't translate:

  1. Open browser console (F12 or F12 → Console)
  2. Check for red error messages
  3. Look for "Language changed to:" message when changing language
  4. Check if "Loaded X articles" appears

Common Issues:

Error: "i18n system not properly initialized"
→ i18n.js might not have loaded yet. Try refreshing the page.
Error: "Blog articles not found in i18n"
→ Articles might not be in language files. Check language files.
Articles don't change on language change
→ Language listener might not be registered. Check console for errors.
