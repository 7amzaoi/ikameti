## 🔧 MAKING CHANGES

### To change the main red color:
Open `css/style.css` and find:
```css
--primary: #C62828;       /* Change this */
--primary-dark: #8E1B1B;  /* And this */
--primary-light: #E53935; /* And this */
```

### To change button colors:
The buttons automatically use `--primary` and `--primary-dark` from above.

### To change text colors:
Find in `css/style.css`:
```css
--text-primary: #222222;   /* Main text */
--text-secondary: #555555; /* Secondary text */
```
