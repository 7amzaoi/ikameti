Typography & Text Colors

Primary Text

This is how your main body text appears. Use this for all important content and headlines. The dark gray color provides excellent contrast and readability.

Secondary Text

Secondary text is used for supporting information, descriptions, and body copy. It maintains good contrast while being visually distinct from primary text.

Tertiary Text

Use tertiary text for muted information like captions, timestamps, and supplementary content. Still readable but clearly less emphasized.

🔴 Accent Text (Links, CTAs)

This red color is used for links, buttons, and interactive elements. It draws attention and indicates actionable items. Hover state: #8E1B1B
