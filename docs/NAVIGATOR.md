## 🚀 Getting Started

### For Website Visitors
Simply open **[index.html](index.html)** in a browser

### For Developers
1. Read [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md)
2. Check [docs/README.md](docs/README.md) for full documentation
3. Look at [docs/QUICK_START.md](scripts/QUICK_START.md) for development tasks

### To Add Content
- **New blog article**: Edit [blog/data/articles.json](blog/data/articles.json)
- **New language**: Create new file in [lang/](lang/) (e.g., de.json)
- **New page**: Create .html file in root, update navigation

### To Debug
- Blog issues: See [scripts/BLOG_DEBUG.js](scripts/BLOG_DEBUG.js)
- i18n issues: Check [docs/I18N_SYSTEM.md](docs/I18N_SYSTEM.md)
- Design issues: Review [docs/DESIGN_SYSTEM.md](docs/DESIGN_SYSTEM.md)
